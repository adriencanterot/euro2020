# Euro 2020 Pronostics

Proof of Concept (< 1 Jour de code) de site de pronostique entre collègue pour l'#euro2020 utilisant :

-   [Next.js](https://nextjs.org/)
-   [ChakraUI](https://chakra-ui.com/)
-   [CMS by Strapi](https://strapi.io)
-   Deployed on [Vercel](https://vercel.com/) & [Heroku](https://heroku.com/)

## Démarrer

Server dev

```bash
npm run dev

yarn dev
```

Ouvrir [http://localhost:3000](http://localhost:3000).
